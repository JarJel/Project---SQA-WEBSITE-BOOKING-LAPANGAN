# White Box Testing Documentation

Folder ini berisi seluruh rancangan kasus uji, skenario, metode, dan tinjauan pengujian **White Box Testing** (Pengujian Kotak Putih). Pengujian ini berfokus pada struktur internal program, alur eksekusi logika, kondisi percabangan, perulangan, dan penulisan kode program secara langsung.

## Dokumen di dalam folder ini:

1.  **[README.md](file:///c:/SQA-PROJECT/3_Software_Testing_Documentation/White_Box_Testing/README.md)**: File panduan ini.
2.  **[Basis_Path_Testing.md](file:///c:/SQA-PROJECT/3_Software_Testing_Documentation/White_Box_Testing/Basis_Path_Testing.md)**: Pengukuran kompleksitas siklomatis (*cyclomatic complexity*) dan pendefinisian jalur independen dari kode program.
3.  **[Code_Walkthrough.md](file:///c:/SQA-PROJECT/3_Software_Testing_Documentation/White_Box_Testing/Code_Walkthrough.md)**: Dokumentasi proses tinjauan kode informal secara tim (*code walkthrough sessions*).
4.  **[Control_Flow_Testing.md](file:///c:/SQA-PROJECT/3_Software_Testing_Documentation/White_Box_Testing/Control_Flow_Testing.md)**: Pengujian alur kendali berdasarkan cakupan pernyataan (*Statement Coverage*) dan percabangan (*Branch Coverage*).
5.  **[Data_Flow_Testing.md](file:///c:/SQA-PROJECT/3_Software_Testing_Documentation/White_Box_Testing/Data_Flow_Testing.md)**: Pengujian siklus hidup variabel berdasarkan relasi *Definition-Use Chains*.
6.  **[Desk_Checking.md](file:///c:/SQA-PROJECT/3_Software_Testing_Documentation/White_Box_Testing/Desk_Checking.md)**: Simulasi manual penelusuran logika program baris demi baris (*Dry Run Table*).
7.  **[Formal_Inspection.md](file:///c:/SQA-PROJECT/3_Software_Testing_Documentation/White_Box_Testing/Formal_Inspection.md)**: Tinjauan formal penemuan cacat logika kode (*Fagan Inspection*) dengan struktur pertemuan formal.
8.  **[Loop_Testing.md](file:///c:/SQA-PROJECT/3_Software_Testing_Documentation/White_Box_Testing/Loop_Testing.md)**: Pengujian perulangan sistem (seperti perulangan cek ketersediaan slot) untuk mendeteksi kesalahan batas nilai (*boundary values*).
